# 占星书籍深度拆解 Skill 设计文档

## 概述

创建一个专门用于拆解占星书籍的 skill（`astrology-book-skill`），每次拆解产出两份独立文件：中文深度拆解 + 英文深度拆解（原创写作，非翻译）。

## 需求总结

- 仅产出"深度拆解"模块，不需要 Obsidian 增强版和精简版
- 中文版和英文版内容结构完全独立，各自面向母语读者
- 占星书籍细分为 6 种子类型，每种有独立模板
- 配套专属验证脚本

## 文件结构

```
~/.claude/skills/astrology-book-skill/
├── SKILL.md                        # 主 skill 文件
├── references/
│   ├── type-templates-cn.md        # 中文版 6 种子类型模板
│   ├── type-templates-en.md        # 英文版 6 种子类型模板
│   └── methodologies.md            # 分析方法论（复用+扩展）
└── scripts/
    └── validate.sh                 # 双语验证脚本
```

## 占星子类型（6 种）

| 子类型 | 识别特征 | 示例书籍 |
|--------|---------|---------|
| 心理占星 | 荣格心理学、原型、无意识、人格发展 | Liz Greene 系列、Sasportas《The Twelve Houses》 |
| 演化占星 | 灵魂演化、南北交点、前世业力、冥王星 | Jeffrey Wolf Green《Pluto》、Steven Forrest《Yesterday's Sky》 |
| 入门教程 | 基础概念教学、星盘解读入门 | 《The Inner Sky》、Arroyo 入门系列 |
| 行运与预测 | 行运、推运、时间技法、生命周期 | Robert Hand《Planets in Transit》 |
| 占星哲学与历史 | 哲学基础、思想史、学术研究 | Tarnas《Cosmos and Psyche》、Campion 历史系列 |
| 神话与原型 | 神话象征、行星原型、超个人心理学 | 《Mythic Astrology》、Butler《The Archetypal Universe》 |

## 输出规范

### 文件路径
- 中文版：`~/obsidian/占星书/中文/《书名》 - 作者.md`
- 英文版：`~/obsidian/占星书/英文/《Book Title》 - Author.md`

### 格式
两版均为纯净 Markdown（无 YAML、无 callouts、无 wikilinks、无 ==高亮==）

### 中文版特点
- 第二人称"你"
- 占星术语首次出现时附英文原文（如"行运（transit）"）
- 类型自适应语气

### 英文版特点
- 原创英文写作，使用标准占星英文术语
- 独立的章节结构和侧重点
- 语气：knowledgeable but accessible

### 字数范围

| 子类型 | 中文字数 | 英文词数 |
|--------|---------|---------|
| 心理占星 | 8000-12000 | 4000-6000 |
| 演化占星 | 8000-12000 | 4000-6000 |
| 入门教程 | 5000-8000 | 2500-4000 |
| 行运与预测 | 7000-10000 | 3500-5000 |
| 占星哲学与历史 | 10000-15000 | 5000-7500 |
| 神话与原型 | 8000-12000 | 4000-6000 |

## 工作流

1. 识别占星子类型（6选1）
2. 评估复杂度，确定篇幅
3. 生成中文深度拆解
4. 生成英文深度拆解（独立写作）
5. 运行验证脚本 → 修复循环（最多3轮）
6. 内容质量快检

## 验证脚本

```bash
bash validate.sh <中文文件> <英文文件> <占星子类型>
```

### 中文版检查项
- 文件存在 + 命名格式 + 纯净 MD
- 子类型必需章节 + 字数范围
- 写作规范（方法论零泄露、格式净化、第二人称、加粗密度）

### 英文版检查项
- 文件存在 + 命名格式 + 纯净 MD
- 子类型必需章节（英文） + 词数范围
- 写作规范英文版（无方法论泄露、加粗密度）

## 复用的核心原则

- 方法论完全隐形
- 格式净化规则（消除 AI 痕迹）
- 防幻觉 + 防遗漏
- 掌握度过低时的声明
