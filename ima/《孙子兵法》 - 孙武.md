# 《孙子兵法》深度读书笔记

## 一、元信息/坐标定位

### 作者背景

**孙武（约公元前545年-约公元前470年）**，字长卿，春秋时期齐国人，后奔吴国，被吴王阖闾任命为将。他是中国古代最伟大的军事家、战略家，被后世尊为"兵圣"。相传他以《孙子兵法》十三篇见吴王，并通过训练宫女展示了他的治军之道，从此深受重用。

### 写作背景

《孙子兵法》成书于春秋末期（约公元前512年），正值中国历史上战争频繁、诸侯争霸的时代。这一时期的战争规模扩大、战术复杂化，对军事理论提出了更高要求。孙武总结了前人的军事经验，结合自己的实战心得，创作了这部划时代的著作。

### 学科定位

本书是**军事战略学的奠基之作**，同时横跨哲学、政治学、管理学等多个领域。它不仅是一部兵书，更是一部关于**竞争哲学和决策智慧**的经典。其思想被广泛应用于商业竞争、体育竞技、外交谈判等非军事领域。

### 核心价值

《孙子兵法》的核心价值在于：它超越了具体的战术层面，提炼出**关于竞争与冲突的普遍智慧**。2500年后的今天，它依然是世界上被研究最多的战略著作，被译成几十种语言，影响了从拿破仑到现代商业领袖的无数决策者。

## 二、全书逻辑地图

本书的核心逻辑结构如下：

**战争的本质：国之大事**

**战前准备**
- 计篇：五事七计
- 作战篇：速战速决
- 谋攻篇：不战而胜

**战略层面**
- 形篇：先胜后战
- 势篇：造势用势
- 虚实篇：避实击虚

**战术层面**
- 军争篇：以迂为直
- 九变篇：灵活应变
- 行军篇：相敌之术
- 地形篇：因地制宜
- 九地篇：置之死地

**特殊作战**
- 火攻篇：特殊手段
- 用间篇：情报战争

**全书逻辑主线**：十三篇构成一个**从战略到战术的完整体系**。前三篇讨论战前的评估与准备，确立"先胜而后求战"的核心理念；中间六篇展开形、势、虚实等核心战略概念；后四篇涉及具体的战场态势和特殊作战方式。全书贯穿着**以最小代价获取最大胜利**的思想，强调计算、准备、灵活和信息的重要性。

## 三、核心主题/逐章深度拆解

### 第一篇：计篇——战争的理性评估

**核心命题**："兵者，国之大事，死生之地，存亡之道，不可不察也。"
战争是关乎国家生死存亡的大事，必须进行严肃的理性评估。

**五事：战略评估的五个维度**

| 要素 | 含义 | 现代诠释 |
|------|------|----------|
| **道** | 政治正义性，民心向背 | 使命愿景、企业文化、员工认同 |
| **天** | 天时，自然条件 | 市场时机、行业周期、宏观环境 |
| **地** | 地利，地理条件 | 市场位置、渠道优势、地缘因素 |
| **将** | 将领才能 | 领导力、管理团队素质 |
| **法** | 军法制度 | 组织架构、流程制度、激励机制 |

**七计：比较优劣的七个问题**

孙子提出了著名的**七计**来比较敌我双方：

1. 主孰有道？（哪方更得民心）
2. 将孰有能？（哪方将领更有才能）
3. 天地孰得？（哪方占天时地利）
4. 法令孰行？（哪方制度执行更好）
5. 兵众孰强？（哪方军队更强）
6. 士卒孰练？（哪方训练更精）
7. 赏罚孰明？（哪方赏罚更分明）

**核心洞察**：**知胜有五**：知可以战与不可以战者胜；识众寡之用者胜；上下同欲者胜；以虞待不虞者胜；将能而君不御者胜。这五条至今仍是判断竞争优势的金标准。

**诡道十二法**

"兵者，诡道也" ——战争是欺骗的艺术。孙子列举了十二种欺敌之术：能而示之不能，用而示之不用，近而示之远，远而示之近，利而诱之，乱而取之，实而备之，强而避之，怒而挠之，卑而骄之，佚而劳之，亲而离之。

**战略思维**：这些诡道的核心是**信息不对称的利用**。在竞争中，让对手误判你的意图、能力和位置，是获取优势的重要手段。

### 第二篇：作战篇——战争的经济学

**核心命题**："兵贵胜，不贵久。"
战争最可怕的是旷日持久，速战速决是基本原则。

孙子详细计算了战争的成本：

- **日费千金**：每天的军费开支惊人
- **百姓之费**：战争对民生的消耗
- **国家财力**：长期战争会掏空国库

**经济战争观**：孙子是第一个系统论述**战争经济学**的思想家。他指出："故不尽知用兵之害者，则不能尽知用兵之利也。"不理解战争的代价，就无法真正理解如何用兵。

**因粮于敌**：

孙子提出了"因粮于敌" 的思想——从敌方获取补给。"食敌一钟，当吾二十钟"——从敌人那里获取一钟粮食，相当于从本国运来二十钟。

**现代应用**：在商业竞争中，这意味着**利用对手的资源**：挖角竞争对手的人才、学习对手的成功经验、进入对手已经培育好的市场。

### 第三篇：谋攻篇——不战而胜的艺术

**核心命题**："是故百战百胜，非善之善者也；不战而屈人之兵，善之善者也。"
打一百场胜仗不是最高明的，不战而让敌人屈服才是最高明的。

这是《孙子兵法》最核心的思想之一。孙子建立了一个**攻击层次的优先级**：

1. **上策：伐谋** - 通过战略威慑瓦解对方
2. **其次：伐交** - 通过外交手段孤立对方
3. **其次：伐兵** - 通过战场交战击败对方
4. **下策：攻城** - 通过攻城消耗己方

**核心洞察**："攻城之法，为不得已。" 直接攻坚是最后的选择，因为代价最高、损失最大。真正的战略高手追求的是**用最小的代价获得最大的胜利**。

**全胜思想**：

孙子提出了**全胜**的概念："故善用兵者，屈人之兵而非战也，拔人之城而非攻也，毁人之国而非久也。必以全争于天下。"

这里的"全"有三层含义：
1. 保全自己（减少损失）
2. 完整获取（不破坏敌人的资源）
3. 速战速决（不长期消耗）

### 第四篇：形篇——先胜后战的逻辑

**核心命题**："胜兵先胜而后求战，败兵先战而后求胜。"
胜利的军队是先确保能赢然后才开战，失败的军队是先开战然后企图获胜。

孙子在这里阐述了**形**的概念。"形"指的是客观的实力对比，是可以测量、计算的硬实力。

**先为不可胜**：

**防守哲学**："昔之善战者，先为不可胜，以待敌之可胜。不可胜在己，可胜在敌。"
从前善于用兵的人，首先创造自己不可被战胜的条件，然后等待可以战胜敌人的机会。不被战胜取决于自己，能否战胜敌人取决于对方。

这是一个深刻的战略思想：**你只能控制自己，不能控制敌人**。所以战略的第一步是让自己立于不败之地。

**胜于易胜**：

孙子强调："善战者之胜也，无智名，无勇功。"真正善战的人取得胜利时，既没有料事如神的名声，也没有勇猛杀敌的战功。为什么？因为他们**只打已经确定能赢的仗**。

**核心洞察**："故善战者，立于不败之地，而不失敌之败也。是故**胜兵先胜而后求战**，败兵先战而后求胜。"这是全书最重要的句子之一。

### 第五篇：势篇——创造压倒性优势

**核心命题**："凡战者，以正合，以奇胜。"
一般作战，用正面力量交战，用出奇制胜的方法取胜。

如果说"形"是静态的实力，"**势**"则是动态的优势。孙子用几个比喻说明"势"：

- **激水之疾，至于漂石者，势也**（湍急的水流能漂走石头，这就是势）
- **鸷鸟之疾，至于毁折者，节也**（猛禽俯冲的速度能折断猎物的骨头，这就是节）

**奇正相生**：

孙子提出了**奇正**的辩证法："凡战者，以正合，以奇胜。故善出奇者，无穷如天地，不竭如江河。"

| 正 | 奇 |
|----|----|
| 常规部署 | 出其不意 |
| 正面对抗 | 侧翼突击 |
| 预期之中 | 意料之外 |
| 稳定基础 | 制胜关键 |

**关键理解**：奇正不是固定的，而是**相互转化**的："奇正相生，如循环之无端，孰能穷之？"今天的奇招，明天可能变成常规；今天的常规，换个情境可能就是奇招。

**势的构建**：

孙子讲如何构建势："故善战者，求之于势，不责于人。故能择人而任势。"——善战者追求的是创造有利态势，而不是苛求士兵。他们选择合适的人然后利用态势。

**现代应用**：在商业中，**势**意味着：品牌优势、网络效应、规模经济、先发优势。好的战略是先构建势，然后借势取胜，而不是让员工硬拼。

### 第六篇：虚实篇——集中优势力量

**核心命题**："兵之形，避实而击虚。"
用兵的规律是避开敌人坚实之处，攻击敌人空虚之处。

这是《孙子兵法》中最具战术价值的一篇。核心思想是**集中优势兵力，攻击敌人弱点**。

**致人而不致于人**：

**主动权**："故善战者，致人而不致于人。"
善战的人能调动敌人而不被敌人调动。这是关于**主动权**的论述——永远要掌握主动，让敌人跟着你的节奏走。

**虚实的辩证法**：

孙子详细讨论了如何创造虚实对比：

- 能使敌人自至者，利之也（用利益诱使敌人来到你想要的地方）
- 能使敌人不得至者，害之也（用威胁阻止敌人去到他想去的地方）
- 故敌佚能劳之，饱能饥之，安能动之（敌人休息时让他疲劳，敌人吃饱时让他饥饿，敌人安定时让他移动）

**形人而我无形**：

**信息优势**："故形人而我无形，则我专而敌分。"
让敌人暴露形态而我隐藏形态，这样我可以集中而敌人分散。孙子在2500年前就深刻理解了**信息不对称**的战略价值。

### 第七篇：军争篇——争夺主动权

**核心命题**："军争之难者，以迂为直，以患为利。"
军队争夺主动权最困难的是把迂回变成直达，把不利变成有利。

孙子讨论了军队机动中的核心挑战：如何在争夺有利位置时，既要快速又要保持战斗力。

**以迂为直**：

表面上看起来绕远的路，可能实际上是最快的路。因为直路上可能有敌人的阻击，迂回的路虽然远但没有阻碍。

**现代应用**：在商业竞争中，**以迂为直**意味着：
- 不正面与巨头竞争，而是先占领细分市场
- 不直接进入饱和市场，而是创造新品类
- 不拼价格，而是差异化竞争

**风林火山**：

孙子著名的**风林火山**出自本篇：

"故其疾如风，其徐如林，侵掠如火，不动如山，难知如阴，动如雷震。"

| 状态 | 特征 | 应用 |
|------|------|------|
| 疾如风 | 快速机动 | 抓住机会迅速行动 |
| 徐如林 | 整齐有序 | 维持组织纪律 |
| 侵掠如火 | 猛烈进攻 | 进入市场要坚决 |
| 不动如山 | 稳定防守 | 守住核心阵地 |
| 难知如阴 | 隐蔽意图 | 保护商业机密 |
| 动如雷震 | 出击震撼 | 产品发布要有冲击力 |

### 第八篇至第十一篇：战术层面的智慧

#### 九变篇——灵活应变

**核心思想**："将通于九变之利者，知用兵矣。"
将领如果精通各种变化，就懂得用兵了。

孙子强调将领必须具备灵活应变的能力，不能拘泥于固定的规则。他提出了著名的**五危**——将领的五种危险性格：

1. 必死（有勇无谋，可以被诱杀）
2. 必生（贪生怕死，可以被俘虏）
3. 忿速（急躁易怒，可以被激怒）
4. 廉洁（过于爱惜名誉，可以被侮辱）
5. 爱民（过于爱护百姓，可以被骚扰）

#### 行军篇——判断敌情

本篇讨论行军中如何**相敌**（观察判断敌情）。孙子列举了大量细节，如：

- 远而挑战者，欲人之进也（远远挑战是想诱你前进）
- 辞卑而益备者，进也（言辞谦卑却加强战备的，是要进攻）
- 半进半退者，诱也（一半前进一半后退的，是诱饵）

#### 地形篇——因地制宜

孙子分析了六种地形（通、挂、支、隘、险、远）和六种失败（走、弛、陷、崩、乱、北），强调将领必须理解地形对作战的影响。

**核心洞察**："知彼知己，胜乃不殆；知天知地，胜乃可全。"
这是对"知己知彼"的扩展——不仅要了解敌我，还要了解天时地利。

#### 九地篇——置之死地而后生

**核心思想**："投之亡地然后存，陷之死地然后生。"
把军队置于绝境然后才能保存，陷入死地然后才能求生。

孙子分析了九种战略地形（散地、轻地、争地、交地、衢地、重地、圮地、围地、死地），并讨论了在每种地形上应采取的策略。

最著名的是**置之死地而后生**的思想：当军队没有退路时，反而会爆发出最强的战斗力。"吾士无余财，非恶货也；无余命，非恶寿也。"士兵抛弃财物不是不爱财，准备拼命不是不爱命，而是形势所迫。

### 第十二篇：火攻篇——特殊手段

本篇讨论火攻的五种类型和实施条件。更重要的是，孙子在篇末发出了著名的警告：

**战争的审慎**："主不可以怒而兴师，将不可以愠而致战。合于利而动，不合于利而止。怒可以复喜，愠可以复悦，亡国不可以复存，死者不可以复生。故明君慎之，良将警之。"
国君不能因为愤怒而发动战争，将领不能因为恼恨而挑起战斗。这是全书最重要的警句之一。

### 第十三篇：用间篇——情报的价值

**核心命题**："故明君贤将，所以动而胜人，成功出于众者，先知也。"
明智的君主和贤能的将领之所以能够战胜敌人，是因为他们预先获取了情报。

孙子在最后一篇强调了**情报**的极端重要性。他提出了五种间谍（因间、内间、反间、死间、生间），并指出：

**情报投资**："故三军之事，莫亲于间，赏莫厚于间，事莫密于间。"
在军队事务中，没有比间谍更亲近的，没有比间谍赏赐更丰厚的，没有比间谍的事情更机密的。

这说明孙子深刻理解**信息的价值**。他愿意在情报上投入最多的资源，因为好的情报可以避免无数的损失。

## 四、核心框架提炼

### 框架一：孙子的战略金字塔

从优到劣的战略选择：
1. **伐谋（最优）** - 战略层面瓦解敌人
2. **伐交** - 外交层面孤立敌人
3. **伐兵** - 战场层面击败敌人
4. **攻城（最差）** - 强攻消耗

### 框架二：知胜五要素

| 要素 | 内容 | 判断问题 |
|------|------|----------|
| **知战** | 知道何时该打何时不该打 | 这场仗值得打吗？ |
| **识众寡** | 理解兵力多少的运用之道 | 我有资源优势吗？如何使用？ |
| **上下同欲** | 全军上下目标一致 | 团队是否齐心？ |
| **以虞待不虞** | 有准备等待无准备的敌人 | 我准备好了吗？对手准备好了吗？ |
| **将能君不御** | 将领有才能且不被过度干预 | 执行者是否有能力和自主权？ |

### 框架三：形势虚实的辩证体系

三组核心概念：
- **形**（静态实力）vs **势**（动态优势）
- **奇**（出其不意）vs **正**（正面对抗）
- **虚**（薄弱环节）vs **实**（强固之处）

战略的艺术在于：用**形**构建基础，借**势**创造优势，以**奇**打破僵局，以**正**稳定战线，**避实击虚**，集中优势。

## 五、批判性思考

### 5.1 本书的贡献

1. **系统性**：第一部完整的军事战略理论体系
2. **抽象性**：从具体战术上升到一般原则
3. **辩证性**：强调变化、转化和具体情境
4. **人文性**："慎战"思想体现了对战争的深刻反思

### 5.2 本书的局限

批判性反思：
1. **时代局限**：基于冷兵器时代的经验，有些内容不完全适用于现代战争
2. **道德争议**：诡道思想可能导致不择手段的倾向
3. **过度应用**：商业不完全是战争，过度军事化思维可能有害
4. **确定性假设**：孙子假设可以"知胜"，但现实中存在大量不可知的因素

### 5.3 与其他战略思想的对话

- 与克劳塞维茨《战争论》对比：孙子强调智胜，克劳塞维茨强调意志和摩擦
- 与《君主论》对比：马基雅维利更关注政治统治，孙子更关注军事战略
- 与博弈论对比：孙子的很多思想可以用博弈论语言重新表述
- 与《道德经》对比：两者都强调柔弱胜刚强、以退为进

### 5.4 在商业中的应用边界

《孙子兵法》在商业中的应用需要注意边界：

1. 商业竞争与战争的本质不同：商业更多是正和博弈，战争更多是零和博弈
2. "不战而胜"在商业中意味着：差异化定位、蓝海战略、创造新市场
3. 过度竞争思维可能忽略了合作共赢的可能性

## 八、费曼终极检验

**如果要向一个高中生解释这本书的核心内容，我会这样说：**

《孙子兵法》是2500年前一个中国将军写的关于打仗的书，但它讲的道理适用于所有竞争。

**第一个道理：打仗之前先算账**。不是热血上头就冲上去，而是冷静算一算：我们准备好了吗？对方什么情况？时机对吗？条件允许吗？算清楚了再行动。

**第二个道理：最好的胜利是不战而胜**。如果你能让对手自己放弃，或者让他压根就不想跟你打，那比拼个你死我活强一百倍。在商业里，这意味着找一个没人跟你竞争的位置，而不是在红海里厮杀。

**第三个道理：只打能赢的仗**。真正厉害的人不是打一百场胜仗，而是只打那些他确定能赢的仗。"先胜后战"——先确保能赢，然后才开打。

**第四个道理：打到痛处**。不要跟对手最强的地方硬碰硬，而是找他的弱点集中攻击。这叫"避实击虚"。

**第五个道理：信息比武力更重要**。谁掌握的信息更多、更准确，谁就更可能赢。所以孙子说，间谍是最值得投资的。

**一句话总结：聪明人不是打赢一百场仗，而是只打那些不得不打、而且能赢的仗。**
