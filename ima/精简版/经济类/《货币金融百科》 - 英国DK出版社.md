# 《货币金融百科》精简版

## 本书定位

这是DK出版社的图文百科作品，用视觉化方式呈现金融领域的全景图谱。追求"广度"而非"深度"，适合零基础读者构建完整的货币与金融认知框架。

---

## 核心知识点

### 一、货币基础

**货币的三大功能**：
1. 交换媒介：解决以物易物的双重需求巧合问题
2. 价值尺度：提供统一的价格表达方式
3. 价值储藏：跨越时间转移购买力

**现代货币创造机制**：现代货币主要不是央行"印"出来的，而是商业银行通过贷款创造的。

货币乘数效应：央行发行100元基础货币，在20%准备金率下，最终可创造500元广义货币。

### 二、银行体系

**商业银行本质**：借短贷长的金融中介，赚取期限错配的利差。

三大风险：
- 流动性风险：存款可随时提取，贷款是长期的
- 信用风险：借款人可能无法偿还
- 利率风险：利率变动影响资产负债价值

**中央银行四大职能**：货币政策、金融稳定、银行的银行、政府的银行

**货币政策工具**：政策利率、公开市场操作、准备金要求、量化宽松

### 三、金融市场

**股票**：公司所有权凭证，享有分红权、投票权和资本增值潜力

估值方法：市盈率（P/E）、市净率（P/B）、DCF估值

**债券**：债务凭证，到期必须偿还，收益相对固定

收益率曲线：正常形态（长期>短期）、倒挂形态（衰退信号）

**外汇**：全球最大金融市场，日交易量超6万亿美元

不可能三角：固定汇率+资本自由流动+独立货币政策，三者不可兼得

**衍生品**：期货、期权、掉期、远期

双面性：风险管理vs杠杆放大

### 四、个人理财

**理财金字塔**：
- 底层（保障）：应急基金、保险
- 中层（增值）：指数基金、债券、房产
- 顶层（投机）：个股、期权、加密货币

**核心原则**：
1. 先保障，后增值
2. 分散投资
3. 长期持有
4. 成本意识

**72法则**：资金翻倍年数 = 72 / 年化收益率

**行为偏差**：损失厌恶、处置效应、羊群效应、过度自信、锚定效应

### 五、金融危机

**危机循环模式**：信贷扩张 - 资产泡沫 - 泡沫破裂 - 信贷紧缩 - 经济衰退 - 政策救助

**2008年危机教训**：复杂性本身就是风险。当没人真正理解金融产品时，灾难就在酝酿。

### 六、金融科技

金融科技图谱：支付、借贷、理财、保险、区块链、监管科技

CBDC vs 加密货币：
- CBDC：央行发行，国家信用背书
- 加密货币：去中心化，价值来自共识

## 核心框架

### 金融体系五层架构

| 层次 | 内容 | 关键概念 |
|------|------|---------|
| 货币层 | 交换媒介 | 法定货币、通胀 |
| 银行层 | 金融中介 | 存贷利差、信用创造 |
| 市场层 | 价格发现 | 股票、债券、外汇 |
| 监管层 | 系统稳定 | 央行、资本要求 |
| 科技层 | 效率提升 | 支付、智能投顾 |

### 金融健康检查清单

1. 应急基金：3-6个月开支的流动储备？
2. 保险保障：医疗、意外、重疾覆盖？
3. 负债管理：信用卡每月还清？
4. 投资配置：根据年龄和目标做资产配置？
5. 退休规划：在积累养老金？

### 市场周期四阶段

1. 复苏期：股票领涨
2. 繁荣期：商品表现好
3. 滞胀期：现金为王
4. 衰退期：债券避险

## 一句话总结

金融系统如同人体血液循环：货币是血液，银行是心脏，金融市场是血管网络，央行是大脑。理解金融不是为了成为投资大师，而是为了在这个金融化的世界里不被收割。时间和复利是普通人最强大的武器，但前提是足够早地开始。
