插图主题：六阶段消息流水线
风格：blueprint — 精密工程蓝图风格

视觉构图：
- 主视觉：水平流水线图，6 个处理节点从左到右依次排列
- 布局：
  - 每个节点是一个蓝色边框的圆角矩形，内部包含：
    ① 几何图标 + 中文名称 + 英文名称
  - 6 个节点：
    1. 漏斗图标 — "摄入 / Ingestion" — 接收原始消息
    2. 锁头图标 — "访问控制 / Access Control" — 验证权限
    3. 对话气泡图标 — "会话解析 / Session Resolution" — 路由到正确会话
    4. 拼图图标 — "上下文组装 / Context Assembly" — 构建完整上下文
    5. AI 芯片图标 — "模型调用 / Model Invocation" — 发送到 LLM
    6. 发送箭头图标 — "响应投递 / Response Delivery" — 返回结果
  - 节点之间用带箭头的粗直线连接，箭头用工程蓝
- 顶部标题："消息处理六阶段流水线 / Six-Stage Message Pipeline"
- 每个节点下方有一行简要说明文字

配色方案：
- 主色：工程蓝 #2563EB（边框、箭头、图标）
- 背景：蓝图纸白底 #FAF8F5
- 节点填充：浅蓝 #BFDBFE
- 文字：深灰 #334155
- 强调节点（④上下文组装）：使用深蓝 #1E3A5F 边框加粗

文字内容：
- 每个节点中英双语标注
- 节点下方简要说明

风格要点：严格水平排列，等间距，管道风格的工程流程图。连接线带箭头，保持直角和对齐。
