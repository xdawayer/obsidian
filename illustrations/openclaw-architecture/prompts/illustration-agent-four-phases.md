插图主题：Agent 执行四阶段
风格：blueprint — 精密工程蓝图风格

视觉构图：
- 主视觉：四个蓝图模块从左到右排列，每个模块内列出关键子步骤
- 布局：
  - Phase 1 — 上下文组装 / Context Assembly：
    蓝色边框模块，内部列出子步骤图标：
    · 加载对话历史
    · 注入系统提示词
    · 检索相关记忆
    · 加载 Skills/Scenario
  - Phase 2 — 模型推理 / Model Inference：
    蓝色边框模块，内部：
    · 流式传输到 LLM
    · 增量 Token 输出
    · 检测 tool_calls
  - Phase 3 — 工具执行 / Tool Execution：
    蓝色边框模块，内部：
    · 运行时拦截 tool_call
    · 沙箱环境执行
    · 结果回注上下文
  - Phase 4 — 状态持久化 / State Persistence：
    蓝色边框模块，内部：
    · JSONL 保存会话
    · 更新记忆文件
    · 触发通知/自动化
  - 模块间用带箭头粗线连接
- 底部标注循环箭头："Agent Loop — 思考 → 行动 → 观察 → 持久化"

文字内容：
- 四个阶段中英双语标注
- 每个阶段内 3-4 个子步骤
- 底部循环标注
