插图主题：工具系统全景
风格：blueprint — 精密工程蓝图风格

视觉构图：
- 主视觉：中心是 Agent 核心圆形节点，8 个工具组以辐射状排列
- 布局：
  - 中心：标注 "Agent Core / Agent 核心" 的圆形蓝色节点
  - 8 个工具组以 45 度间隔辐射排列，每个工具组是一个矩形框：
    1. 终端图标 — runtime / 运行时命令
    2. 文件夹图标 — fs / 文件系统
    3. 地球图标 — web / 网络请求
    4. 浏览器图标 — ui / 用户界面
    5. 信封图标 — messaging / 消息通信
    6. 多人图标 — sessions / 会话管理
    7. 大脑图标 — memory / 记忆操作
    8. 齿轮图标 — automation / 自动化
  - 每个工具组用直线连接到中心
- 底部分离区域：展示 Function Calling 执行流程
  - 模型输出 → 运行时拦截 → 权限检查 → 沙箱执行 → 结果回注
  - 权限检查标注五级权限："Global → Provider → Agent → Sandbox → Owner-only"
  - 右侧警告框："拒绝列表永远优先 / Deny list always wins"
