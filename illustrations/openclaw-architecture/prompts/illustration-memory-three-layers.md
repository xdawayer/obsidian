插图主题：三层记忆架构
风格：blueprint — 精密工程蓝图风格

视觉构图：
- 主视觉：三层堆叠的蓝图剖面，表示记忆层级
- 布局：
  - Layer 1（底层/基础层）— 持久记忆 / Persistent Memory：
    最大的底层框，内部显示 MEMORY.md 文件图标
    标注："策划的长期知识 / Curated Long-term Knowledge"
    标签："手动编辑 + AI 追加"
  - Layer 2（中层）— 临时记忆 / Ephemeral Memory：
    中等大小的框，内部显示日历 + 文件图标
    标注："memory/YYYY-MM-DD.md 每日日志"
    标签："自动生成，7天后过期"
  - Layer 3（顶层）— 会话记忆 / Session Memory：
    最小的顶层框，内部显示聊天气泡图标
    标注："sessions/*.jsonl 对话记录"
    标签："完整对话历史"
  - 三层从下到上堆叠，用尺寸线标注层级关系
- 右侧独立面板：混合搜索引擎
  - 向量搜索 Vector Search (权重 0.7)
  - + BM25 关键词搜索 (权重 0.3)
  - → MMR 重排序
  - → 时间衰减加权
  - 底部标注："SQLite + sqlite-vec"

- 搜索面板：浅灰背景
- 文字：深灰 #334155

文字内容：
- 三层中英双语标注
- 搜索引擎参数说明
- 底部技术栈标注

风格要点：层叠剖面图，越底层越深色表示越持久。右侧搜索面板用细线连接到三层，表示可搜索所有层级。
