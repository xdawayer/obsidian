视觉构图：
- 主视觉：时间线/流程图，从左到右展示记忆刷写机制
- 布局：
  - 左侧 — 触发条件：
    一个长条进度条表示上下文窗口 (Context Window)
    标注 "176K / 200K tokens"
    填充到 ~88%，最后 20% 用琥珀色(#F59E0B)标注为红线警戒区
    标注："达到 80% 阈值 → 触发"
  - 中间 — 静默代理回合 (Silent Agent Turn)：
    一个蓝色虚线框，内部显示 AI 图标正在"提取"
    箭头从上下文窗口指向右侧文件
    标注操作：
    · 提取关键信息
    · 识别重要决策
    · 保留核心上下文
  - 右侧 — 双结果：
    上方：memory/ 文件夹图标，标注 "关键信息已安全存储"
    下方：压缩后的上下文窗口（较短的进度条），标注 "旧消息被安全压缩"
    虚线箭头从 memory 回到新上下文，标注 "语义搜索随时召回"
- 底部对比框（两列）：
  - 左列（灰色/划掉）："传统 AI：压缩 = 失忆"
  - 右列（蓝色/高亮）："OpenClaw：压缩前刷写 = 永不失忆"
